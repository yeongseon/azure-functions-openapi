# SUPPORT.md

Support & Compatibility Policy

## Supported Environment

* Python: **3.10 and above**
* Runtime: Azure Functions (Python)

---

## Support Scope

This project is maintained by a **single developer**.

Support is provided on a **best-effort basis**:

* Bug reports with clear reproduction steps are welcome
* Feature requests may be accepted if aligned with project goals
* Questions about Azure infrastructure or deployment are out of scope

---

## Deprecation Policy

* Deprecated behavior will be documented before removal
* Deprecations are announced at least **one minor release** in advance
* Removal occurs in a future major or minor version as appropriate

---

## Security Issues

* Security issues should be reported privately if possible
* Fixes will be released as soon as reasonably possible

See docs/SECURITY.md for reporting details.

---

## Non-Goals

The following are explicitly not supported:

* Guaranteed response times
* Custom feature development
* Environment-specific debugging
