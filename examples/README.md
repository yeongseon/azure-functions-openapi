# Examples

`azure-functions-openapi` keeps two smoke-tested examples:

| Role | Path | Description |
| --- | --- | --- |
| Representative | `examples/hello_openapi` | Minimal HTTP trigger plus OpenAPI JSON, YAML, and Swagger UI routes. |
| Complex | `examples/todo_crud_api` | CRUD-style app with multiple operations, Pydantic models, and generated spec output. |
