# Demo Assets

The README demo assets are generated from code:

- `run_hello_openapi_example.py` executes the representative `hello_openapi` example flow and generates the OpenAPI and Swagger UI artifacts used by the README.
- `openapi-cli.tape` renders the terminal GIF with VHS from that same example execution.
- `make demo` regenerates both the terminal GIF and the Swagger UI preview image.
