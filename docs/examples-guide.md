## How to Use the Examples

1. Start with `docs/examples/hello_openapi.md` to learn the basics.
2. Follow `docs/examples/todo_crud_api.md` for a complete CRUD flow.
3. Use the trigger-specific examples for integration patterns.

Each example includes a minimal code sample and notes about configuration.

## Downloadable Starter Templates

You can create a starter template from any example directory:

```bash
git archive --format=zip --output hello_openapi.zip HEAD examples/hello_openapi
```

This command produces a zip file you can share or import into a new project.
